✨ Key Features
👑 Admin Panel
Full user management (Admins, Instructors, Students, Parents)

Course approval & category management

Platform analytics & revenue reports

Subscription plan management (Monthly/Yearly)

👨‍🏫 Instructor Dashboard
Course creation (video, document, SCORM/xAPI)

Assignment & coding challenge creation

Online class scheduling (live classes)

Quiz builder (MCQs, coding problems, subjective)

Student progress tracking & grading

👩‍🎓 Student Dashboard
Browse & enroll in courses (free/paid)

Complete assignments & coding challenges

Take quizzes with instant feedback

Join online live classes

Earn points, badges & certificates

Track learning progress & achievements

👪 Parent Portal
Monitor child's course progress

View quiz scores & assignment completion

Track certificates earned

💰 Payment & Subscription
Monthly & Yearly subscription plans

Secure payment gateway integration

Automatic invoice generation

🎮 Gamification
Points system for completed tasks

Badges for milestones

Certificates upon course completion
